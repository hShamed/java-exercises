# Second Exercise

## Instructions

Write a Java program that uses a `for` loop to print the numbers from 1 to 22, each on a new line.

### Requirements

- Use a `for` loop to iterate from 1 to 22.
- Print each number on its own line using `System.out.println`.

### Example Output

```
1
2
3
...
...
...
22
```

Good luck!
